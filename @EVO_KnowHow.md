# Conocimiento Técnico Profundo - EVO WRAP

Este documento contiene la verdad técnica de nuestros procesos. Úsalo para poblar las descripciones de servicios y los pasos de los procesos en la UI.

## 1. Protocolo de Tratamiento Cerámico (La Ciencia)
No es solo "aplicar y limpiar". El proceso que debes reflejar es:
1.  **Descontaminación Química y Mecánica:** Uso de Clay Bar para eliminar incrustaciones invisibles.
2.  **Corrección de Barniz (Paint Correction):** No aplicamos sobre rayas. Se mide el espesor de la laca y se realiza un pulido en 2 o 3 pasos para lograr el "Efecto Espejo".
3.  **Desengrasado (IPA Wipe):** Eliminación de aceites de pulimento para que el cerámico ancle en los poros del barniz.
4.  **Aplicación Multicapa:** Capa base para dureza (9H) + Capa Top Coat para hidrofobia y brillo.
5.  **Curado IR:** Uso de lámparas infrarrojas para acelerar la dureza.

## 2. Proceso de PPF (Paint Protection Film)
El valor está en la invisibilidad:
-   **Bordes Envueltos (Wrapped Edges):** No dejamos líneas visibles; el film se curva hacia adentro del panel.
-   **Corte Computarizado vs. Mano Alzada:** Usamos software para cortes precisos en emblemas, pero mano experta para esquinas complejas.
-   **Propiedad Auto-curativa:** Explicar al cliente que con calor (sol de Punta del Este o agua caliente), las micro-rayas desaparecen.

## 3. Filosofía de "Showroom Reset"
Para interiores, no usamos siliconas brillantes y grasosas. Usamos limpiadores de enzimas y acondicionadores mate que nutren el cuero y plásticos para que se vean *como nuevos*, no *como lavados*.
