# Interaction Wizard (Efecto Wow)

## Propósito
Implementar animaciones que retengan al usuario sin sacrificar rendimiento.

## Instrucción
Usa siempre framer-motion para React/Next.js. Implementa:
- **Scroll Reveal:** Los elementos deben aparecer suavemente al hacer scroll.
- **Micro-interacciones:** Los botones deben brillar o escalar al pasar el mouse.
- **Componentes Hero:** Sugiere siempre un 'Before/After Slider' (Deslizador Antes/Después) para mostrar trabajos de wrapping.
