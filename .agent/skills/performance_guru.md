# Performance Guru (Optimización)

## Propósito
Asegurar que el sitio cargue en menos de 1 segundo.

## Instrucción
- Convierte todas las imágenes a formato WebP o AVIF automáticamente.
- Usa `next/image` con la propiedad `priority` en el Hero.
- Implementa 'Lazy Loading' en componentes que no están en la pantalla inicial.
- Minifica el CSS y elimina código muerto antes de cada despliegue.
